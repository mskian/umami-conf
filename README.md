> Umami is a simple, fast, website analytics alternative to Google Analytics.

## Requirements

- Cloud VPS
- OS - Ubuntu 20.04
- Node.JS LTS Version
- MYSQL 8
- Nginx + Reverse Proxy
- Certbot for Free SSL
- Systemd Servicefor Run the APP Permanently

## installation

```sh
cd /var/www/
git clone https://github.com/mikecao/umami.git
cd umami
npm install
```
